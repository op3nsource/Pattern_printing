# Can you code these Triangle patterns with python


### Pattern 1

input : 5
```
*
* *
* * *
* * * *
* * * * *
```

<details> <summary>Show Solution</summary>

Solution by : [@Sanky-coder](https://github.com/Sanky-coder)

```python
for i in range(1,6):
    for j in range(1,i+1):
        print("*",end="")
    print("\n")
```
